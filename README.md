# DoggoSnap - Fetch Dog Photos with Ease! (AND101 Lab 5)

DoggoSnap is a fun Android app that lets you effortlessly fetch and browse through random dog photos. Powered by API calls, DoggoSnap brings a constant supply of adorable doggos right to your fingertips. Plus, with the help of the Glide library, viewing and caching these cute dog images is smoother than ever. Get ready to smile with every swipe as you discover a world of furry friends with DoggoSnap!
